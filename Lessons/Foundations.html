2.2 Setting Up Kali Linux on a Virtual Machine (VM)

Step 1 : Download Required Files

  • Kali Linux ISO: Download Kali Linux ISO
  • VirtualBox: Download VirtualBox
  • VMware: Download VMware Workstation Player

Step 2 : Install VirtualBox & Create a VM

Create a Virtual Machine

  • Open VirtualBox and click New
  • Name it "Kali Linux" and select Type: Linux, Version: Debian (64-bit)
  • Allocate 4GB+ RAM and 2+ CPU Cores
  • Create a 20GB+ Virtual Hard Disk (VDI format, Dynamically Allocated)
  • Select the Kali Linux ISO as the boot disk

Step 3: Start Kali Linux Installation

  • Start the VM and select Graphical Install
  • Set up language, region, and keyboard
  • Configure hostname (e.g., "kali")
  • Set up username and password
  • Choose Guided â€“ Use Entire Disk (for VM)
  • Complete the installation and reboot
